🎬 Movie Recommendation System

A Java-based Movie Recommendation System that suggests Bollywood movies based on genre or minimum rating.

Features

- Recommend movies by genre
- Recommend movies by rating
- Console-based interface
- Uses Java OOP concepts and ArrayList

Technologies Used

- Java
- OOP
- ArrayList
- Scanner

Learning Outcomes

- Classes and Objects
- Constructors
- ArrayLists
- User Input Handling
- Filtering Data
